# react

## react是什么？

- JS库 提供了UI层面的解决方案
- 单向数据流
- 组价可以组合嵌套构成完整的页面

特性：

- JSX语法
- 单向数据流
- 虚拟dom

优势：

高效：对 DOM 模拟 不直接操作DOM 

灵活：和已知的

生态丰富

受控组件和非受控组件区别？应用场景？

受控组件：组件状态（value）由React的state完全控制

非受控组件：组件的状态由DOM自身控制，React不控制其value，需要通过ref从dom节点获取数据。

应用场景

受控组件适合需要实时控制，效验或修改表单的数据的场景：

如：实时表单验证，表单数据联动

非受控组件：

适合**无需实时控制、仅需最终提交数据**的简单场景

如：简单的表单提交（如单个输入框的搜索、登录按钮，仅需点击时获取值）；

文件上传（`<input type="file" />`只能是非受控组件，因为其 value 只读，无法通过 state 设置）；

减少性能开销
