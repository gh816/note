表格合并方案

整体表格的数据结构

```
{
	cells: Cell[][],
}
```

单元格

每个单元格必须包含的字段

```
{
	id: string,
	content: string，
	rowSpan:number;向下合并
	colSpan:向右合并
	isMerged
}
```

合并区域

合并区域（`merge(startRow, startCol, endRow, endCol)`）

实际区域[r1, c1]` 到 `[r2, c2]

主体单元格：

rowSpan = r2 - r1 + 1;

colspan = c2 -c1 + 1

isMerged = false;

遍历区域内所以其他单元格（r，c）:

设置 `isMerged = true`

渲染表格

遍历cells

如果cell.isMerged === true 跳过渲染
